# postgrey

Ansible role for postgrey, the postfix greylisting policy server.

## Requirements

A running postfix instance.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

    - hosts: servers
      roles:
        - postgrey

or

    - hosts: servers
      roles:
        - { role: postgrey, x: 42 }

if any variables comes in the future for this role.
